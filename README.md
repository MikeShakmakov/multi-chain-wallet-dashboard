# Frontend

This project was generated using [Angular CLI](https://github.com/angular/angular-cli) version 21.2.2.

## Libraries

Material, tailwind, apollo

## Architecture

Layers: component, entity(business logic), form(form handling), GQL(network requests + conversions), RXJS for state management/requests, signals for templates

All code was written manually except one of the last commits - transactions feature, as all examples was prepared and only copy paste development left.
